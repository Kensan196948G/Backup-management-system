# WebUI サンプル仕様書 INDEX

## 概要

本フォルダは、バックアップ管理システムの **サンプルWebUI** に関するすべての種類の仕様書を格納しています。
各仕様書はそれぞれの目的に応じて分類されており、開発・設計・運用の各フェーズで参照できます。
